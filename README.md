State of the world is climate change tracker for the top 15 economies by GDP.

Frontend consists of high performannt 3D globes for visualization.

Backend consists of a serverless API for fetching data from OpenWeatherMap and storing in MongoDB. Sentiment analysis is done with a tiny bag of words model trained on Hrishik's [Climate Change News](https://kaggle.com/fringewidth/climate-change-news).

The stack was initally heavily reliant on Azure Language Studio, Azure Web Apps and Azure Cosmos DB. After running out of student credits, it was refactored to run purely in the free tier of Vercel.
